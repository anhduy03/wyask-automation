## Key Tiers

The Wyask platform supports multiple user tiers:

1. **Freemium** - Free tier with limited features
2. **Plus** - $12/month subscription tier
3. **Pro** - Premium subscription tier
4. **ProT (Pro Teams)** - Team collaboration tier

Each tier has specific limits for queries, envelopes, storage, and recipients.
